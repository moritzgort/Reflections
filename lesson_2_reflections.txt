What happens when you initialize a repository? Why do you need to do it?

You create it, you need to do this in order to track different versions of your file(s).

***

How is the staging area different from the working directory and the repository? What value do you think it offers?

The staging area is something that takes a file to prepare it for the working directory.

***

How can you use the staging area to make sure you have one commit per logical change?

You can add your files from the working directory to the staging directory and from there on commit them. You can add a message that describes what you changed.
